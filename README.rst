=========================
Read Comic Online scraper
=========================

Setup
=====

From the command line::

    git clone https://github.com/Girbons/comics-scraper

Requirements
============

From the command line::

    pip install -r requirements/requirements.txt

Usage:
=======

From the command line::

    cd comics

.. code-block:: python

    python downloader.py -u ['comic url']
